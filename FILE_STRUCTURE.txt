🌐 CONTEXT PROVIDERS
--------------------
CartContext.jsx       - Shopping cart state and functions
                        • addToCart, removeFromCart, updateQuantity
                        • getCartTotal, getCartCount, clearCart
                        • Persists to localStorage

AuthContext.jsx       - User authentication state
                        • login, register, logout, updateProfile
                        • Mock authentication (replace with real API)
                        • Persists user to localStorage

-------
products.js           - Product catalog with 16 sample products
                        • 6 categories (laptops, desktops, etc.)
                        • Product details, images, specs, tags
                        • Helper functions for filtering and search

                        • Hero slider with 3 slides
                        • Featured products section
                        • Category grid
                        • Promotional banners
                        • Newsletter signup

                        • Advanced filtering (category, price, rating, tags)
                        • Sorting options
                        • Search functionality
                        • Responsive filters sidebar

                        • Large product image
                        • Product specifications
                        • Add to cart with quantity selector
                        • Related products
                        • Breadcrumb navigation

                        • Cart items list
                        • Quantity controls
                        • Price calculations (subtotal, tax, shipping)
                        • Promo code input
                        • Checkout button

Register.jsx          - User registration

Account.jsx           - User dashboard
