🐛 TROUBLESHOOTING
================================================================================

Problem: Site still shows old version after 10 minutes
Solution: 
  1. Clear browser cache (Ctrl + Shift + Delete)
  2. Try incognito/private browsing
  3. Check Actions tab for errors

Problem: GitHub Actions workflow failing
Solution:
  1. Go to Settings → Actions → General
  2. Select "Read and write permissions"
  3. Check "Allow GitHub Actions to create and approve pull requests"
  4. Save and re-run workflow

Problem: 404 errors on the site
Solution:
  1. Make sure you enabled "GitHub Actions" as source
  2. Wait for deployment to complete
  3. Check that base path is '/modern-ecommerce/' in vite.config.js

Problem: Blank page
Solution:
  1. Open browser console (F12)
  2. Look for errors
  3. Check that all files deployed correctly in Actions tab
