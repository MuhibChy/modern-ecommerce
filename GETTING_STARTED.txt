================================================================================
                         TROUBLESHOOTING
================================================================================

ISSUE: PowerShell script execution blocked
FIX: Use Command Prompt instead of PowerShell
     OR run: Set-ExecutionPolicy RemoteSigned -Scope CurrentUser

ISSUE: npm command not found
FIX: Install Node.js from https://nodejs.org/

ISSUE: Port 3000 already in use
FIX: Run: npx kill-port 3000
     OR: npm run dev -- --port 3001

ISSUE: Installation very slow
FIX: Use different npm registry
     Run: npm config set registry https://registry.npmjs.org/

ISSUE: Module not found errors
FIX: Delete node_modules and package-lock.json
     Run: npm install
